## Описание
GUI-приложение на Python/Tkinter для генерации паролей с настройкой длины (ползунок 4-64), типов символов (чекбоксы), отображением пароля и историей в таблице/JSON.

## Установка и запуск
1. Установите Python 3.
2. `pip install tkinter` (обычно встроено).
3. Запустите `python password_generator.py`.

## Использование
- Настройте длину ползунком.
- Выберите символы чекбоксами.
- Нажмите "Генерировать".
- История сохраняется в `history.json`.

## Примеры
- Длина 12, буквы+цифры: `aB3cD5eF7gH9`
- С спецсимволами: `K#9m@2P$4q%`
